## 維護規則

- 中文內容**零破折號**（U+2014、U+2013）。台灣繁體中文。
- 改完 `git push` 後約 1 分鐘 GitHub Pages 自動上線。
- 每個 case study 都內建「列印完整內容」按鈕，產生 PDF 由本人 Cmd+P 自行存檔。
- `@media print` CSS 會在列印時隱藏 tab 與返回鍵，並展開所有章節。
